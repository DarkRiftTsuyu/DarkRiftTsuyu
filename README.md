# Projects

## **Checkpoint**
A website / app that lets users add friends to compete with each other in 100% completion races of any game (primarily single-player).  
Users can use Steam to search for a game to automatically get a race list of all achievements or create a blank race to add checkpoints themselves.

**Find it here:** https://checkpoint-race.app
